Device Features & Controls

Master every feature of your nuyu device

Control Panel Overview

The nuyu device features an intuitive touch-sensitive control panel with the following elements:

Main Display

  • Current time (large digits)
  • Next alarm time
  • Active mode indicator
  • Scent level indicators

Touch Controls

  • Mode Button: Cycle through sleep, wake, and manual modes
  • Intensity +/-: Adjust scent diffusion strength
  • Light Button: Control ambient lighting
  • Snooze/Stop: Large button for alarm control

Operating Modes

Sleep Mode

Activated 30 minutes before your set bedtime:

  • Gradually increases calming scent
  • Dims lights to warm red tones
  • Reduces display brightness
  • Activates do-not-disturb

Wake Mode

Begins at your set wake time:

  • Releases energizing scent
  • Gradually brightens lights
  • Plays gentle wake sounds
  • Increases intensity if no movement detected

Manual Mode

For custom control:

  • Direct control over both scent wells
  • Adjustable light color and intensity
  • Timer function for meditation or focus sessions

Advanced Features

Smart Wake

Uses motion detection to start your wake routine within a 30-minute window when you're in lighter sleep.

Weekend Mode

Automatically adjusts your schedule by up to 1 hour on weekends while maintaining consistency.

Travel Mode

Quickly adjust to new time zones with accelerated schedule shifting.
